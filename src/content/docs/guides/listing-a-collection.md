◉RD only uses parent/child or galleries for collection data, relying strictly on on-chain data.

To submit a previously inscribed gallery (or parent, if parent/child provenance was used when
creating your collection) to [ord.net](https://ord.net/) for verification and trading, we use
[GitHub](https://github.com/). If you do not already have a GitHub account, please
[create a free account](https://github.com/signup) before continuing.

When you are ready, follow these steps:

1. Go to [TheWizardsOfOrd/ordinals-collections issues](https://github.com/TheWizardsOfOrd/ordinals-collections/issues).
2. Click **New Issue**.
3. Click **Submit Collection**.
4. Add a title.
   - This should match the title of your gallery.
5. Set **Collection Type** to **gallery**, unless you are submitting a new collection that uses
   parent/child provenance, in which case select **parent**.
6. Add the single gallery inscription ID, or one parent inscription ID per line.
7. Add the slug for your collection.
   - Do not use spaces or capitalization.
8. Add any additional info that is relevant to your submission.
9. Click **Create**.

After your gallery or parent is verified, it will be open for trading on
[ord.net](https://ord.net/).
